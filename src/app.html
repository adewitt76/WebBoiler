This is a minimal webpack boilerplate project that gives a basic template to build a web project.

How to use

 — Delete the .git file
 — Rename root directory
 — Change name in package.json
 — Change title in webpack.common.js
 — Install any webpack modules or plugins needed for the project

Build and Run

If you would like to run a hot dev server run watch and the server simultaneously.

 — Build(development): npm run build
 — Build(production): npm run build-prod
 — Dev Server: npm start
 — Watch: npm run watch

Possible Loaders:

* babel loader to ensure backwards compatability
* TypeScript loader
